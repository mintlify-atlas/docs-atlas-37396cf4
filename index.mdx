      Get up and running with Dioxus in minutes
    </p>
  </div>

  <Steps>
    <Step title="Install the Dioxus CLI">
      Install the Dioxus CLI tool using cargo:

      ```bash
      cargo install dioxus-cli
      ```

      The CLI provides hot-reload, bundling, and cross-platform building capabilities.
    </Step>

    <Step title="Create a new project">
      Use the CLI to scaffold a new Dioxus project:

      ```bash
      dx new my-app
      cd my-app
      ```

      This creates a new project with all the necessary configuration files.
    </Step>

    <Step title="Run your app">
      Start the development server with hot-reload:

      ```bash
      dx serve
      ```

      Your app will open in a window. Edit your code and see changes instantly without losing state.
    </Step>
  </Steps>
</div>
