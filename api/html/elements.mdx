Dioxus provides a comprehensive set of HTML elements that can be used with the `rsx!` macro. All elements are type-safe and come with comprehensive documentation.

## Document Metadata

### base

The `<base>` element specifies the base URL to use for all relative URLs in a document.

```rust
rsx! {
    base { href: "https://example.com/", target: "_blank" }
}
```

**Attributes:** `href`, `target`

**Reference:** [MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/base)

### head

The `<head>` element contains machine-readable information (metadata) about the document.

```rust
rsx! {
    head {
        title { "My App" }
        meta { charset: "utf-8" }
    }
}
```

**Reference:** [MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/head)

### link

The `<link>` element specifies relationships between the current document and an external resource.

```rust
rsx! {
    link { 
        rel: "stylesheet", 
        href: "/styles.css",
        r#type: "text/css"
    }
}
```

**Attributes:** `href`, `rel`, `type`, `crossorigin`, `hreflang`, `media`, `sizes`, `title`, `integrity`, `disabled`, `referrerpolicy`, `fetchpriority`, `blocking`, `as`

**Reference:** [MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/link)

### meta

The `<meta>` element represents metadata that cannot be represented by other HTML meta-related elements.

```rust
rsx! {
    meta { charset: "utf-8" }
    meta { name: "viewport", content: "width=device-width, initial-scale=1" }
    meta { property: "og:title", content: "My App" }
}
```

**Attributes:** `charset`, `content`, `http_equiv`, `name`, `property`

**Reference:** [MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/meta)

### style

The `<style>` element contains style information for a document.

```rust
rsx! {
    style { r#type: "text/css", ".my-class {{ color: red; }}" }
}
```

**Attributes:** `type`, `media`, `nonce`, `title`

**Reference:** [MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/style)

### title

The `<title>` element defines the document's title shown in the browser's title bar or page tab.

```rust
rsx! {
    title { "My Page Title" }
}
```

**Reference:** [MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/title)

## Sectioning Root

### body

The `<body>` element represents the content of an HTML document.

```rust
rsx! {
    body {
        h1 { "Welcome" }
        p { "Content here" }
    }
}
```

**Reference:** [MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/body)


**Reference:** [MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/address)

## Forms

### form

The `<form>` element represents a document section containing interactive controls.

```rust
rsx! {
    form { 
        action: "/submit",
        method: "post",
        input { r#type: "text", name: "username" }
        button { r#type: "submit", "Submit" }
    }
}
```

**Attributes:** `action`, `autocomplete`, `enctype`, `method`, `name`, `novalidate`, `target`

**Reference:** [MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/form)

### button

The `<button>` element represents a clickable button.

```rust
rsx! {
    button { 
        r#type: "button",
        onclick: |_| println!("Clicked!"),
        "Click me"
    }
}
```

**Attributes:** `autofocus`, `disabled`, `form`, `formaction`, `formenctype`, `formmethod`, `formnovalidate`, `formtarget`, `name`, `type`, `value`

**Reference:** [MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button)

### input

The `<input>` element creates interactive controls for forms.

```rust
rsx! {
    input {
        r#type: "text",
        name: "username",
        placeholder: "Enter username",
        value: "{username}",
        oninput: move |e| username.set(e.value())
    }
}
```

**Attributes:** `accept`, `alt`, `autocomplete`, `autofocus`, `capture`, `checked`, `disabled`, `form`, `height`, `list`, `max`, `maxlength`, `min`, `minlength`, `multiple`, `name`, `pattern`, `placeholder`, `readonly`, `required`, `size`, `src`, `step`, `type`, `value`, `width`

**Input Types:** `button`, `checkbox`, `color`, `date`, `datetime-local`, `email`, `file`, `hidden`, `image`, `month`, `number`, `password`, `radio`, `range`, `reset`, `search`, `submit`, `tel`, `text`, `time`, `url`, `week`

**Reference:** [MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input)

### label

The `<label>` element represents a caption for a form control.

```rust
rsx! {
    label { r#for: "username", "Username:" }
```

**Attributes:** `for`, `form`

**Reference:** [MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/label)

### select

The `<select>` element represents a control for selecting among options.

```rust
rsx! {
    select { 
        name: "choice",
        onchange: move |e| selection.set(e.value()),
```

**Attributes:** `autocomplete`, `autofocus`, `disabled`, `form`, `multiple`, `name`, `required`, `size`, `value`

**Reference:** [MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/select)

### option

## Element Usage

All elements follow the same pattern in `rsx!`:

```rust
rsx! {
    element_name {
        // Attributes first
        attribute_name: "value",
        another_attribute: true,
        
        // Then children
        "Text content"
        child_element {}
    }
}
```
